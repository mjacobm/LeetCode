# LeetCode Solutions

This repository contains C# solutions to various LeetCode problems.

## Problem Solutions

The repository includes solutions to the following problems:

- **TwoSum** - Find two numbers that add up to a target sum
- **ValidParentheses** - Validate if parentheses are correctly matched
- **LongestSubstringWithoutRepeatingCharacters** - Find the longest substring without repeating characters
- **MaximumSubarraySum** - Find the maximum sum of a contiguous subarray
- **ReverseAString** - Reverse a string
- **FibonacciNumber** - Calculate Fibonacci numbers
- **MergeIntervals** - Merge overlapping intervals
- **ProductOfArrayExceptSelf** - Calculate product of array elements except self

## Structure

Each problem solution is implemented in its own C# file with a class containing the solution method(s).

## Running the Solutions

This is a C# project using .NET. To run:

1. Open the solution file (`LeetCode.slnx`) in Visual Studio or your preferred C# IDE
2. Build the project
3. Run the `Program.cs` file to test the solutions

## Requirements

- .NET Framework (as specified in `LeetCode.csproj`)
- Visual Studio or another C# IDE
